# CLAUDE.md — 项目约束与规范

## 数据安全（最高优先级）

- **严禁在代码、文档、README、注释、测试用例中使用任何真实业务数据**，包括但不限于：
  - 真实广告项目名称、广告单元名称、推广计划名称
  - 真实消耗金额、展示量、点击量、转化量等投放数据
  - 真实广告主 ID、APP_ID、APP_SECRET、Token
  - 真实公司名称、客户名称、活动名称
- 所有示例数据必须使用明显的虚构占位值（如"示例项目-品牌推广-通投"、`1234567890`、`your_app_id_here`）
- 提交前必须检查 diff 中是否包含真实业务数据

## .env 与凭证

- `.env`、`token.json` 等含凭证的文件已在 `.gitignore` 中，绝不可提交
- 示例配置只能使用占位符，不得包含任何真实值

## 项目结构

- Go 语言项目，使用巨量引擎 Marketing API
- 单二进制 CLI 工具
